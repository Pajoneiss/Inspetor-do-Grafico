# Ladder Labs | Premium Dashboard (Vercel)

This is the high-performance, rich-aesthetic dashboard for the Inspetor trading engine.

## Deployment to Vercel

1. **Connect Repository:** Log in to [Vercel](https://vercel.com) and click "Add New Project".
2. **Select Repo:** Choose `Inspetor-do-Grafico`.
3. **Configure Project:**
   - **Root Directory:** Set this to `apps/dashboard-premium`.
   - **Framework Preset:** Next.js.
4. **Environment Variables:**
   - Add `NEXT_PUBLIC_API_URL`: Set this to your Railway URL (e.g., `https://inspetor-do-grafico-production.up.railway.app`).
5. **Deploy:** Click Deploy.

## Design Features
- **Extreme Rounding:** 32px (`radius-xl`) and 24px (`radius-lg`) borders.
- **Glassmorphism:** All cards use backdrop-blur for a premium feel.
- **Micro-animations:** Integrated with `framer-motion` for fluid state transitions.
- **Liquid Theme:** Ultra-dark (#050505) with neon green accents.
